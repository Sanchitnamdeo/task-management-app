# 🚀 TaskFlow Pro - Professional Task Management Application

[![Node.js](https://img.shields.io/badge/Node.js-18.x-green.svg)](https://nodejs.org/)
[![Socket.io](https://img.shields.io/badge/Socket.io-4.x-blue.svg)](https://socket.io/)
[![SQLite](https://img.shields.io/badge/SQLite-3.x-blue.svg)](https://sqlite.org/)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

A real-time task management application built with **Node.js**, **Express**, **Socket.io**, and **SQLite**. Experience seamless task management with instant updates across all connected clients.

---

## ✨ Features

### 🔐 Authentication
- User Registration with email verification
- Secure Login with JWT tokens
- Session Management
- Password Encryption (bcrypt)

### 📋 Task Management
- ✅ Create, Read, Update, Delete tasks
- 🔄 Real-time task updates via WebSockets
- 🏷️ Task Priority (Low, Medium, High)
- 📊 Task Statistics Dashboard
- 🔍 Filter tasks by status (All/Active/Completed)
- ✏️ Inline task editing
- 🗑️ Bulk delete completed tasks

### 🎨 User Interface
- Modern, responsive design
- Glass-morphism effects
- Smooth animations
- Mobile-friendly
- Real-time connection status
- Professional color scheme

### ⚡ Real-time Features
- Instant task updates across all devices
- Live connection status indicator
- Automatic reconnection
- Broadcast updates to all connected users

## 🛠️ Technologies Used

### Backend
- **Node.js** - JavaScript runtime
- **Express.js** - Web framework
- **Socket.io** - Real-time WebSocket library
- **SQLite3** - Lightweight database
- **bcryptjs** - Password hashing

### Frontend
- **HTML5** - Structure
- **CSS3** - Styling with animations

- **Font Awesome** - Icons
- **Google Fonts** - Typography

### Development Tools
